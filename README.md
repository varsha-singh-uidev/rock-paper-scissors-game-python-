# Rock Paper Scissors (Python)
A simple command-line Rock, Paper, Scissors game where you play against the computer in a 5-round match.

# Features
* Player vs Computer
* 5 rounds per match
* Score tracking
* Tie gives both players 1 point
* Replay option

# How to Run
* python rock_paper_scissors.py

# Rules
* Same choice → Tie (+1 each)
* Win → +1 point
* Lose → 0 points
* Highest score after 5 rounds wins
